环境搭建

环境搭建

public class HelloWorld{
  public static void main(String[] args) {
    System.out.println("Hello World!");
  }
}
  • 公开的类名必需与文件名一样
  • public static void main 为程序的入口

编译程序

$ javac HelloWorld.java

执行程序

$ java HelloWorld
